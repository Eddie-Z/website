1010TIRES

I spent 4 months at 1010tires as a Full Stack Engineer. One of the major problems was that inventory stock was out of date and done manually. I created multiple CRON job scripts to scrape supplier’s website into XLSX format. This data was then fed into the backend to update existing outdated data. In addition, I’ve optimized the uploading new brand/tires process by creating a script to upload massaged excel data into the backend and frontend. I’ve also initiated the project to optimize customer service workflow by integrating Google API into the backend. This was done using C#, Selenium and SQL.
